MySql 用户管理和授权

发表于 2018-11-15 | 更新于 2018-11-16 | 分类于 技术 ， 数据库 ， MySql

用户管理

  • 创建用户
  • 修改密码
  • 查询

授权管理

  • 授予权限
    格式: grant 权限 on 数据库对象 to 用户@主机

    • privileges：操作权限 
      * SELECT(column,column),INSERT,UPDATE,DELETE   
* CREATE,ALTER,DROP    
* INDEX,references    
* create temporary tables,create view    
* create routine,alter routine,execute
    • databasename：数据库名
    • tablename：表名，如果要授予该用户对所有数据库和表的相应操作权限则可用表示，如.*

  • 查看授权

    • 查看自己的授权

    • 查看其他人的授权

  • 撤销授权

    类似于授权，将 grant 改为 revoke, to 改为 from 即可

  • grant, revoke 某用户权限后，该用户只有重新连接 MySQL 数据库，权限才能生效
  • 如果想让授权的用户，也可以将这些权限 grant 给其他用户，需要选项 “grant option“

Win2003下安装MySql 5.6.42

发表于 2018-11-15 | 分类于 技术 ， 数据库 ， MySql

  • 下载 mysql-5.6.42-win32.zip解压到 E:\mysql-5.6.42-win32

    操作系统是 32位Win2003>
    MySql 5.7 会提示“不是有效的 Win32 应用程序”，该版本不适用

  • 设置配置文件

    复制 my-default.ini 到 my.ini,修改 my.ini

将MySql安装目录下 data 下所有文件拷贝到 datadir(E:\mysqldb\data) 下

  • 配置 MySql 为系统服务
    命令符界面下运行

卸载服务用mysqld.exe -remove

  • 配置环境变量

环境变量PATH增加 E:\mysql-5.6.42-win32\bin

  • 设置MySql初始密码

启动MySql服务后，默认用户 root ， 密码为空, 仅本地登录
mysql -uroot连接MySql
